## Linux Cheat Sheet

## File and Folder Navigation Commands

<table style="border: 1px solid">
  <tr>
    <th>Commands</th>
    <th>Function</th>
  </tr>
  <tr>
    <td>ls</td>
    <td>Lists the contents of a directory</td>
  </tr>
  <tr>
    <td>pwd</td>
    <td>Shows the path to the current directory.</td>
  </tr>
  <tr>
    <td>cd</td>
    <td>Directory change</td>
  </tr>
  <tr>
    <td>locate</td>
    <td>Search for a file and display the path.</td>
  </tr>
  <tr>
    <td>find</td>
    <td>Find a file within the current directory.</td>
  </tr>
</table>

## Commands for operations with files and folders

<table style="border: 1px solid">
  <tr>
    <th>Commands</th>
    <th>Function</th>
  </tr>
  <tr>
    <td>mkdir</td>
    <td>Create a directory</td>
  </tr>
  <tr>
    <td>rmdir</td>
    <td>Remove a directory</td>
  </tr>
  <tr>
    <td>rm</td>
    <td>Delete a file</td>
  </tr>
  <tr>
    <td>cp</td>
    <td>Copy a file to another location.</td>
  </tr>
  <tr>
    <td>mv</td>
    <td>Move or rename a file and folder.</td>
  </tr>
  <tr>
    <td>touch</td>
    <td>Create a file</td>
  </tr>
  <tr>
    <td>cat</td>
    <td>Displays the contents of a file.</td>
  </tr>
  <tr>
    <td>file</td>
    <td>Shows the file type.</td>
  </tr>
</table>
